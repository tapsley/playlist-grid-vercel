# Changelog

## 2026-06-29 — Hard Mode Launch

### New Features
- **Hard Mode** — 15×15 nonograms, available to signed-in users starting July 1. 
- **Medals** — Players will now be awarded medals for the solve times. Fastest solvers receive a gold medal, faster than average solvers receive a purple medal. 
- **Medal Notification** — Medals are awarded at midnight MST, and players who earned them will see a notification by the stats icon the next time they play.
- **Leaderboard tab** — Stats modal now has a Leaderboard tab showing players ranked by gold medal count for the current month. Opens as the default tab.
- **Calendar: play past puzzles** — Players can now view and play past puzzles dating back to April 1.
- **Calendar: solve distributions** — Players can view the solve time distribution for any previous puzzle. Their time will be added to the distribution upon solving.
- **Left-hand mode** — Settings toggle that mirrors the control button layout for left-handed players.

### Improvements
- Celebration animation plays faster for hard mode (18ms/cell vs 60ms).
- Four new celebration sequences added: burst, split reveal, diagonal reverse, bottom-up. Rarer patterns promoted to higher probability.
- Control buttons redesigned, fill and x modes now toggle when clicking either one.
